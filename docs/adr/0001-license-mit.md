# ADR 0001: MIT License

## Status

Accepted

## Context

RetOS is intended to be an open source project that can be reused, forked, embedded, and deployed locally.

## Decision

Use the MIT License.

## Consequences

- Adoption is simple for individuals and companies.
- The license is familiar and low-friction.
- Patent protection is not as explicit as Apache-2.0; this is accepted for the initial project goals.
